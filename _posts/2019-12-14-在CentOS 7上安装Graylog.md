**在安装Graylog服务之前，请确保安装和配置以下软件：**
* Java（> = 8）
* MongoDB（> = 2.4）
* Elasticsearch（> = 2.x）

## 安装java jdk
`$ sudo yum install java-1.8.0-openjdk-headless.x86_64`

## 安装pwgen(密码生成器)
`$ sudo yum install epel-releasesudo | yum install pwgen`
